Files:

1) us-states.json - used to plot the US map
2) stateslived.csv - used to aid visualization
2) vizst.csv - contains data shown via visualization (occupancy, price, inbound passengers, temperature)
3) index.html - contains html/javascript code to run the visualization

To run visualization:

Note: must be run on locally, and recommended to be run using Google Chrome
1) In a terminal window, execute the following command:
	python -m http.server 8765 &

2) In a web browser, go the following URL: http://localhost:8765/
